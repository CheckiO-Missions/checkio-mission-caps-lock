Naciśnięcie dolnego "a" zmienia wielkość liter bloku znaków do następnego "a". Te małe "a" nie są drukowane. Wiedząc o tym, można podzielić dany tekst na oddzielne bloki liter za pomocą dolnego "a" bez zachowywania tego "a". Doskonałym narzędziem do tego jest metoda str.split() z separatorem "a". 

... text.split("a")
Co robić dalej?

W celu wykonania dalszych operacji należy wykonać iterację po sekwencji bloków tekstu, uzyskując ich indeksy i same bloki. Proszę wykonać iterację za pomocą pętli for i funkcji enumerate(). 

... for i, v in enumerate(text.split("a"))
Jak wykonać operację na bloku na podstawie jego indeksu?

Proszę zobaczyć. Tekst jest wpisywany z wyłączonym caps lock aż do pierwszego "a". Tak więc pierwszy blok tekstu (proszę pamiętać o indeksowaniu opartym na 0) nie wymaga żadnych zmian. Po włączeniu caps lock, następny blok tekstu musi być pisany wielkimi literami, potem znowu blok bez zmian itd.

Jeśli indeks bloku tekstu jest nieparzysty, musi on być pisany wielkimi literami. Jak to zrobić? Proszę użyć modulo operator, aby odróżnić indeksy parzyste od nieparzystych, dzieląc je przez 2. Proszę użyć metody str.upper(), aby przekonwertować bloki tekstu z indeksami nieparzystymi. 

(v, v.upper())[i % 2] ...

Tutaj używamy przypomnienia indeksu dzielenia modulo i 2 jako indeksu krotki z początkowymi i zmienionymi blokami tekstu. 

(v, v.upper())[i % 2] for i, v in enumerate(text.split("a"))

Powiedział Pan wszystko, czego potrzeba. Proszę użyć powyższej listy jako argumentu metody str.join() z pustym separatorem linii. 

"".join(...)
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Proszę tylko opakować swój kod w wyrażenie funkcyjne. Zaleca się używanie podpowiedzi typu dla danych wejściowych i wyjściowych.

Oto całe rozwiązanie:

def caps_lock(text: str) -> str:

    return "".join((v, v.upper())[i % 2] for i, v in enumerate(text.split("a")))

W tym rozwiązaniu generator iteruje po podanym tekście i daje początkowe lub wielkie litery w zależności od wartości flagi. Wartość flagi zmienia się za każdym razem, gdy "a" zostanie spełnione. 

def caps_lock(text: str) -> str:

    def characters():

        caps = False
        for char in text:
            if char == 'a':
                caps = not caps
            else: 
                yield char.upper() if caps else char

    return ''.join(characters())

Proszę podzielić tekst i otrzymać listę bloków tekstu. Proszę przekonwertować co drugi element listy na wielkie litery. Proszę połączyć wszystkie elementy. 

def caps_lock(text: str) -> str:

    result = text.split("a")
    
    for x in range(1, len(result), 2):
        result[x] = result[x].upper()
        
    return "".join(result)
