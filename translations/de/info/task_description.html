Joe Palooka hat dicke Finger, daher drückt er immer die Feststelltaste, wenn er eigentlich die Taste "a" selbst drücken will. (Wenn Joe versucht, eine akzentuierte Version von "a" einzugeben, die mehr Tastenanschläge benötigt, um die Akzente hervorzuzaubern, ist er in Gegenwart solcher raffinierten Zeichen vorsichtiger ([Shift] + [a]) und drückt die Tasten richtig). Berechnen Sie das Ergebnis, das sich ergibt, wenn Joe den angegebenen Text eintippt, und geben Sie es zurück. Die Taste "Feststelltaste" wirkt sich nur auf die Buchstabentasten von "a" bis "z" aus und hat keine Auswirkungen auf andere Tasten oder Zeichen. es wird davon ausgegangen, dass die Taste "Feststelltaste" zunächst ausgeschaltet ist.

Bei Joes Tastatur ist die Feststelltaste immer ein Großbuchstabe. Das heißt, wenn die Feststelltaste eingeschaltet ist und Joe die Tastenkombination Umschalt + b drückt, wird 'B' (in Großbuchstaben) gedruckt.
