Catch an idea üí°! Split the text into parts by lower "a", make parts with odd index uppercase and join all parts together.

Ok, sounds easy, but what should I start from?

Pressing lower "a" changes the case of the block of characters until the next "a". These lower "a" are not printed. Knowing this, you may split the given text into separate blocks of letters by lower "a" without keeping this "a"-s. The perfect tool for this is str.split() method with separator "a". 

... text.split("a")
What to do next?

For further operations you need to iterate over the sequence of text blocks, getting their indexes and blocks themselves. Do the iteration with for loop and enumerate() function. 

... for i, v in enumerate(text.split("a"))
How to make operation with a block based on its index?

Let's see. The text is typed with caps lock off until the first "a". So the first block of text (remember about 0-based indexing) needs no changes. After activating caps lock, the next block of text must be in uppercase, then again a block without changes etc.

If the index of text block is odd, it must be in uppercase. How to make it? Use modulo operator to distinguish even indexes from odd by dividing them by 2. Use str.upper() method to convert text blocks with odd indexes. 

(v, v.upper())[i % 2] ...

Here we use a reminder of modulo division index and 2 as index of tuple with initial and changed blocks of text. 

(v, v.upper())[i % 2] for i, v in enumerate(text.split("a"))
Cool! Help me to join it all back together.

You said all you need. Use the above list comprehension as argument of str.join() method with empty line separator. 

"".join(...)
So, how my final solution should look like?

You just need to wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def caps_lock(text: str) -> str:

    return "".join((v, v.upper())[i % 2] for i, v in enumerate(text.split("a")))

In this solution a generator iterates over the given text and yields initial or uppercase character depending of the flag value. The flag value changes each time when "a" is met. 

def caps_lock(text: str) -> str:

    def characters():

        caps = False
        for char in text:
            if char == 'a':
                caps = not caps
            else: 
                yield char.upper() if caps else char

    return ''.join(characters())

Split the text and get a list of text blocks. Convert every second element of the list into uppercase. Join all together. 

def caps_lock(text: str) -> str:

    result = text.split("a")
    
    for x in range(1, len(result), 2):
        result[x] = result[x].upper()
        
    return "".join(result)
